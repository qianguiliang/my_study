how to install xfce desktop on ubuntu 14.04
-------------------------------------------

    ```
    sudo apt-get update
    sudo apt-get install xfce4
    ```

how to install kde 4.12 on ubuntu 14.04
---------------------------------------

    ```
    sudo apt-get update
    sudo apt-get install kubuntu-desktop
    ```
